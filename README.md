# Onlineducation
a little project about online education platform（base in ssm）在线教育平台（ssm开发）
#
#### 技术实现
后端技术：
Spring
SpringMVC
Mybatis
MybatisGenerator
PageHelper
Swagger
Druid
FileUpload
log4j
SSO单点登陆
Redis缓存
七牛云储存

前端技术：
layui
jsp
bootstrap
springfreemaker

1. 用户登录登出及注销功能；
2.显示课程和课程的详细信息；
3.实现登录情况下，用户对所选课程的增删改查功能。
#
基于eclipse实现，下载后import就可以
